VFP Registers

The VFPv2 architecture contains 16 double-precision (64-bit) registers (d0-d15) that can be subdivided into 32 single-precision (32-bit) registers (s0-s31). The VFPv2 architecture also contains several system registers as well, which are detailed in the next few sections.
